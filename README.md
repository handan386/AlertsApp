User Registration & Validation
This is a simple iOS app built with Swift that manages user registration with alert feedback. It ensures that all required information is entered correctly before a user can sign up.

Key Features
Form Validation: Checks if username or password fields are empty.
Password Verification: Confirms that both password entries match.
Smart Alerts: Uses a function to show error or success messages to the user.

Logic
The user fills in the registration form.
The app validates the inputs using if-else logic.
If there is a mistake, a pop-up alert explains the issue.
If everything is correct, a success message appears.

Tech Details
Language: Swift
Framework: UIKit
Architecture: MVC (Model-View-Controller)
